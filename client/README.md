# URL Shortener - Frontend (Client)

A modern, responsive user interface for the Simple URL Shortener application.

## Features

- **Fast Development**: Built with Vite and React for lightning-fast HMR and build times.
- **Modern Styling**: Styled using Tailwind CSS for a highly responsive, clean, and beautiful interface.
- **Component-Driven**: Modular components like `URLInput`, `ResultCard`, and `ActivityTable`.
- **Clipboard Support**: Seamless copy-to-clipboard integration with toast notifications.
- **Type Safety**: Fully typed with TypeScript.

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm

### Installation

1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Scripts

- `npm run dev`: Start the development server.
- `npm run build`: Build the frontend for production.
- `npm run preview`: Locally preview the production build.
